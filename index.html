ZCG Checklist Crawler

User-Agent: ZCGTony/1.0

This crawler fetches the printable checklist view for baseball sets so I can analyze them offline. Scope is Baseball, 2020–2023, inserts included. It runs a single request at a time with ~1.5 seconds between calls and only touches the year/set browse pages for discovery plus the printable checklist endpoint for capture. No login areas, no pricing data, and no images or scripts. Output is HTML only, stored locally for personal research.

Behavior

  • Scope: Baseball 2020–2023, inserts included
  • Pacing: single thread, ~1.5 s delay
  • Endpoints: year and set browse for discovery; /PrintCenter.cfm?Report=PrintChecklist&SetID=… for capture
  • Data stored: printable HTML only
